# 🚀 Inglês para Programação

Aplicação web interativa para aprender inglês focado em programação, com rotina estruturada, acompanhamento de progresso e vocabulário técnico essencial.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido para ajudar desenvolvedores a ganhar autonomia na leitura de documentações, códigos e conteúdos técnicos em inglês.

A aplicação combina:
- 📅 organização de estudos
- 📊 acompanhamento de progresso
- 🧠 memorização de vocabulário
- 🔥 consistência diária (streak)

Tudo em uma interface moderna, responsiva e interativa.

---

## ✨ Funcionalidades

- ✅ Rotina diária com checklist interativo  
- 📅 Calendário de estudos com marcação de dias  
- 🔥 Sistema de sequência (streak)  
- 🧠 Vocabulário técnico com progresso visual  
- 🗓️ Organização semanal de estudos  
- 🛠️ Seleção de ferramentas utilizadas  
- 💾 Salvamento automático com `localStorage`  
- 🎨 Interface moderna e responsiva  

---

## 🎯 Objetivo

Ajudar o usuário a:
- Ler documentação em inglês  
- Entender códigos e erros  
- Consumir conteúdo técnico sem depender de tradução  

Tempo estimado: **3 a 6 meses de prática consistente**

---

## 🧠 Como usar

1. Acesse a aplicação no navegador  
2. Siga a rotina diária proposta  
3. Marque as tarefas concluídas  
4. Registre seus dias de estudo no calendário  
5. Acompanhe seu progresso e sequência  
6. Aprenda e revise o vocabulário técnico
